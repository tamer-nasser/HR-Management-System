# HR Management System

An Excel-based HR Management System designed to analyze employee, payroll, and expense data using reports and dashboards.

## Project Overview
This project focuses on organizing and analyzing HR data to support better decision-making.

## Features
- Employee data management
- Payroll analysis
- Expense tracking
- Data visualization dashboards
- Performance reports

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Analysis

## Project Goals
- Improve HR data organization
- Generate useful insights
- Support business decisions
